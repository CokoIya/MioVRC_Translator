# MioTranslator v1.2.1 正式版アップデート

今回の正式版では、翻訳まわりの実用性、UI の見やすさ、そして GitHub Releases 版の使いやすさを中心に整理しました。

## 主な更新

- 翻訳サービスごとにプリセットモデル選択を追加しました。各モデルに対して、速度・翻訳品質・プラグイン適性をその場で確認できます。
- `GLM` 系サービスを追加し、`GPT` / `DeepSeek` / `GLM` / `Qwen` / `Claude` の切り替えを整理しました。
- 翻訳リクエストの流れを見直し、応答速度の改善と不要な token 消費の抑制を行いました。
- マイク前処理にノイズ除去強度を追加し、環境ノイズによる誤反応や無意味な文字列送信を減らせるようにしました。

## UI 改善

- メイン画面と設定画面をよりコンパクトなデスクトップ UI に再調整しました。
- モデル説明カードを追加し、各モデルの使い分けが分かりやすくなりました。
- マイク選択、手動入力、OSC ガイドなどのポップアップを統一デザインに整理しました。
- ポップアップの初期表示位置を画面内に制限し、アプリアイコンも全ウィンドウで統一しました。

## 機能整理

- VRChat ネイティブ OSC では他プレイヤーの chatbox 本文を取得できないため、実用性のない「他人メッセージ反向翻訳」関連の浮動ウィンドウ機能を削除しました。
- 設定画面の文言とサービス表示名を整理し、より中立で分かりやすい表現に統一しました。

## 配布版メモ

- 今回の GitHub Releases 版バージョンは `v1.2.1_release` です。
- 軽量版には `SenseVoice Small` モデルは同梱されません。初回起動時に未配置であれば自動ダウンロードされます。
- 既存ユーザーは設定画面でモデルを再確認してから使い始めることをおすすめします。
