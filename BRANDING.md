# Brand and Unofficial Distribution Policy

This policy explains how the Mio RealTime Translator project name, identity, and official distribution channels may be used.

The source code is licensed under the GNU General Public License v3.0 or later. That license grants broad rights to use, study, modify, and redistribute the code, provided the license terms are followed. It does not grant permission to use this project's names, logos, icons, website materials, release artwork, screenshots, or other brand assets in a way that suggests an unofficial build is official.

## Protected Project Identity

The following project identity assets are reserved by the project maintainer unless separate written permission is granted:

- The names `Mio RealTime Translator`, `Mio Translator`, and confusingly similar names used for a VRChat or game translation tool
- The project logo, application icon, release artwork, and website visual assets
- The official website and download identity: `https://78hejiu.top/`
- Official release names, package names, and installer presentation that imply a build is published by `ここ_Mio`, `酒寄 みお`, or the official Mio RealTime Translator project

## Permitted Uses

You may:

- Refer to this project by name when giving credit, making a comparison, writing documentation, or explaining that your project is based on Mio RealTime Translator
- Fork the source code and publish your modifications under the GPLv3-or-later license
- Use the project name in a factual sentence such as `Based on Mio RealTime Translator` or `Forked from Mio RealTime Translator`
- Link to the official project website or GitHub repository

## Unofficial Builds and Forks

If you distribute a modified build, fork, installer, mirror, or packaged copy, you must make it clear to users that it is not an official Mio RealTime Translator release.

Required practices for unofficial builds:

- Use a distinct product name that is not confusingly similar to `Mio RealTime Translator` or `Mio Translator`
- Replace the official application icon, logo, installer artwork, website artwork, and any other brand assets
- Clearly state that the build is unofficial and is not endorsed by the Mio RealTime Translator project
- Preserve copyright notices, license notices, and a copy of the GPLv3-or-later license
- Provide the complete corresponding source code for the distributed build as required by GPLv3-or-later
- Do not use the official website, update endpoints, download pages, support channels, or community groups in a way that makes the fork appear official

## Prohibited Uses Without Permission

Without separate written permission, you may not:

- Sell, list, or distribute a build that uses the official name, logo, icon, website branding, or release artwork as if it were an official release
- Remove attribution, copyright notices, or license notices
- Misrepresent a modified version as being published, reviewed, supported, or endorsed by the official project
- Use names, icons, package metadata, installer text, screenshots, or website pages that are likely to confuse users about the source of the software
- Use official update feeds or download infrastructure for an unofficial product

## Official Distribution Channels

Official releases are published through channels controlled by the project maintainer, currently including:

- Official website: `https://78hejiu.top/`
- Official GitHub repository and releases: `https://github.com/CokoIya/MioVRC_Translator`

Users should treat builds from other sources as unofficial unless the project maintainer explicitly says otherwise.
