# MioTranslator v1.2.0_Beta3.2 更新说明

本次版本重点围绕桌面端稳定性、多语言支持、GitHub Releases 轻量分发和新手引导体验进行了集中更新。

## 重点更新

- 新增 GitHub Releases 轻量版分发方案：Releases 特供安装包不再内置 SenseVoice Small 模型，首次启动后会自动下载模型，并在主界面底部显示下载与加载进度。
- 修复打包后因模型未随包携带导致的识别不可用问题，改为运行时自动检测模型是否存在，不存在时自动准备。
- 新增俄语识别与翻译支持，并补全相关语言检测、提示词映射和 SenseVoice 语言参数。
- 桌面端前端语言新增简体中文、英语、日语、俄语、韩语；支持首次按 IP / 系统环境自动选择界面语言，用户手动修改后将永久保存，不会被自动检测覆盖。
- Web 下载页同步新增多语言内容，并增加 GitHub Releases 轻量版下载入口，明确说明该版本不内置模型、首次启动会自动下载。

## 桌面端修复与优化

- 修复“发送到 VRC”成功后仍然误弹“当前场景不可用”告警的问题。
- 修复主页目标语言与设置页配置不同步，导致保存后出现异常告警的问题。
- 修复修改设置后必须重启插件才能生效的问题；保存后会自动重建界面并按当前状态重新加载。
- 修复设置保存后主页偶发只渲染一半、底部按钮或文本区消失的问题。
- 修复多语言切换后按钮、状态文字、底部操作区被挤压甚至消失的问题，重新整理了主界面布局与自适应。
- 修复麦克风设备下拉在部分构建中不显示设备名的问题。
- 修复主界面控件在不同语言下的排列不稳定问题，优化了麦克风栏、目标语言栏、插件语言入口和悬浮窗按钮的布局。

## 功能调整

- 主页新增三页式 OSC 引导弹窗，帮助用户在 VRChat 内开启 OSC。
- 主页新增界面语言快捷入口，并保留设置页中的界面语言修改入口，两个位置都可以直接切换。
- 悬浮窗开关移到主页，悬浮窗会默认定位在“翻译至”相关区域旁边。
- `Base URL` 与 `Model` 改为锁定显示，不允许用户手动修改，但仍会按选定后端正确生效。
- 语音识别后端改为锁定唯一模型 `SenseVoice Small`，不再允许切换到其他后端。
- 主文本输入区高度缩小，更贴近 VRChat 144 字聊天场景。
- 设置窗口高度大幅缩短，主要内容通过滚动查看，减少大窗口遮挡。

## 分发与打包

- 发布版本号更新为 `v1.2.0_Beta3.2`。
- GitHub Releases 特供安装包版本号更新为 `V1.2.0_beta3.2_Releases`。
- 打包流程继续保持禁用 UPX，并保留 Windows 版本信息与发布者信息，降低安装包误报概率。

## 已知说明

- GitHub Releases 轻量版不内置 SenseVoice Small，首次启动时会自动下载模型；请保持网络可用，下载完成后即可离线重复使用。
- 百度网盘 / 群文件渠道仍然适合需要完整离线模型包的用户。
