使用说明

五步开始使用

从下载到 VRChat 实时翻译，只需几分钟。

1

下载并安装

点击下方「立即下载安装」按钮，下载并运行安装程序，安装完成后直接打开 MioTranslator.exe 即可。

2

配置翻译后端

启动 MioTranslator.exe，点击右上角 ⚙ 设置，填写你的 API Key。Base URL 与 Model 将按后端自动锁定。

3

在 VRChat 开启 OSC

进入 VRChat，打开 Action Menu，依次点击：

Options → OSC → Enable
4

语音模型已内置，开箱即用

正式版安装包已内置 SenseVoice Small 语音模型，安装后无需额外下载，直接点击「▶ 开始监听」即可离线使用。

5

开始实时翻译

模型加载完成后，对着麦克风说话，翻译结果将自动出现在你的 VRChat 聊天框中。
