使用说明

五步开始使用

从下载到 VRChat 实时翻译，只需几分钟。

1

下载并安装

前往下载页面，下载 MioTranslator-Setup-v1.1.6.exe，双击运行，按向导选择安装路径（默认 %LocalAppData%\Programs\Mio RealTime Translator）。

2

配置翻译后端

启动 MioTranslator.exe，点击右上角 ⚙ 设置，填写你的 API Key 和翻译模型信息。

3

在 VRChat 开启 OSC

进入 VRChat，打开 Action Menu，依次点击：

Options → OSC → Enable
4

首次使用时自动准备语音模型

如果当前安装包未内置 Whisper 模型，首次点击「▶ 开始监听」时会自动下载到当前用户目录。下载完成后即可离线重复使用。

5

开始实时翻译

模型加载完成后，对着麦克风说话，翻译结果将自动出现在你的 VRChat 聊天框中。
