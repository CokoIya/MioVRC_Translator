功能特性

不只是翻译工具，也是玩家的第二个声音

让跨语言玩家能实时交流，也让无言势、口障或不方便开麦的玩家，把文字变成能被队友听见的 AI 声音。

给每一种表达方式留位置

把“我想说”变成游戏里听得见的声音

语音阅读不只是朗读功能。它可以读出翻译后的句子，也可以读出手打文字；配合 MixLine 后，这段 AI 声音会作为麦克风进入游戏。对无言势、口障、社恐或暂时不方便开口的玩家来说，这意味着可以选择一条适合自己的声线，真正参与语音交流。

🎙️

本地语音识别

本地 + 在线识别：中文 / 粤语用 SenseVoice Small，其他语言可选 Web Speech、Qwen3-ASR 或 Gemini Live。Qwen3-ASR 面向全语言，兼顾高精度和速度。可在设置中随时切换。

🌐

多翻译后端

支持 GPT、Qwen、DeepSeek、Gemini、Claude、Kimi、xAI、Mistral 等服务商，自由切换。

💬

VRChat 直发

通过 VRChat 官方 OSC 接口，将翻译内容直接发送到你的聊天框，无需复制粘贴。

🔄

反向翻译 + 同声传译

反向翻译可识别对方语音并显示译文；开启语音阅读后，译文还能通过 MixLine 虚拟麦克风实时读给游戏里的其他玩家。不是只给 VRC，只要游戏能选麦克风，就能搭出同声传译。

✏️

文字输入 + AI 语音阅读

输入文字可以翻译、发送到 VRC，也可以直接用 AI 声音朗读。无言势、口障或不方便开麦的玩家，可以用自己选择的声线把想说的话传进游戏语音。

🔒

隐私优先

无数据收集、无日志记录、无云服务器。每条消息发送结束后立即销毁，完全本地运行。

使用说明

五步开始使用

从下载到 VRChat 实时翻译，只需几分钟。

1

下载并安装

点击下方「立即下载安装」按钮，下载并运行安装程序，安装完成后直接打开 MioTranslator.exe 即可。

2

配置翻译后端

启动 MioTranslator.exe，点击右上角 ⚙ 设置，填写你的 API Key。Base URL 与 Model 将按后端自动锁定。

3

设置游戏输入

VRChat 聊天框发送需要开启 OSC；如果要让游戏听到 AI 语音，请额外安装 MixLine，并在游戏内选择 MixLine 虚拟麦克风。VRChat OSC 路径：

Options → OSC → Enable
4

下载并安装

运行安装包后，程序会按需下载本地 ASR 或 TTS 模型；已下载的模型会被复用，不会重复下载。SenseVoice 约 950 MB，Whisper 约 1.5 GB；在线 ASR 不需要本地模型，可在设置中选择并填写对应 API Key。

5

开始翻译或同传

模型加载完成后，对着麦克风说话即可翻译；开启反向翻译和语音阅读后，译文会被 AI 语音读出并经 MixLine 进入游戏麦克风。
