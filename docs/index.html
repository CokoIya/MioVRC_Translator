使用说明

五步开始使用

从下载到 VRChat 实时翻译，只需几分钟。

1

下载并解压

点击下方下载按钮跳转百度网盘，下载通过网盘分享的完整文件 Mio，提取码为 0601。解压到任意目录后直接运行其中的 MioTranslator.exe。

2

配置翻译后端

启动 MioTranslator.exe，点击右上角 ⚙ 设置，填写你的 API Key 和翻译模型信息。

3

在 VRChat 开启 OSC

进入 VRChat，打开 Action Menu，依次点击：

Options → OSC → Enable
4

首次使用时自动准备语音模型

测试版完整 ZIP 默认内置 SenseVoice Small。若你自行精简压缩包未附带模型，首次点击「▶ 开始监听」时也会自动下载到当前用户目录，下载完成后即可离线重复使用。

5

开始实时翻译

模型加载完成后，对着麦克风说话，翻译结果将自动出现在你的 VRChat 聊天框中。
