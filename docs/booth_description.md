# 🎤 Mio RealTime Translator

**VRChat向けリアルタイム音声翻訳ツール**
あなたの声をその場で翻訳して、チャットボックスへ自動送信。言葉の壁を越えて、世界中のユーザーと話そう。

---

## ✨ できること

### 自分の声を翻訳して送信
マイクに向かって話すだけ。音声を自動でテキスト化し、選んだ言語に翻訳してVRChatのチャットボックスへリアルタイム送信します。

### 相手の声を翻訳して確認（逆翻訳）
VRChat内で聞こえてくる音声をPCのオーディオ出力から取得し、翻訳・表示します。英語や中国語が飛び交う場でも、何を言っているか把握できます。

### 手動入力でも翻訳
声を使わずテキストを直接入力して翻訳することも可能。チャットの補助として活用できます。

---

## 🌍 主な機能

- **ローカル音声認識**  
  FunASR + SenseVoice Small を使用。初回起動時に自動ダウンロード。音声データはPCの外に出ません。

- **多彩な翻訳バックエンド**  
  GPT / DeepSeek / GLM / Qwen / Gemini / Doubao / Claude に対応。自分のAPIキーで利用します。

- **チャットボックス出力形式を選べる**  
  「翻訳文のみ」「原文のみ」「翻訳文（原文）」「原文（翻訳文）」の4形式から選択可能。

- **翻訳スタイル・口調の設定**  
  日常会話・言語交換・ロールプレイなど用途別モード。ナチュラル・キュート・クールなど口調も選択可能。

- **ノイズ除去 + 音声区間検出（VAD）**  
  環境ノイズによる誤検知を抑制。感度や除去強度を細かく調整できます。

- **OSC連携**  
  VRChatのOSC機能に直接対応。アバターパラメーターへの翻訳ステータス連携も設定可能。

- **フローティングウィンドウ**  
  翻訳結果を常に手前に表示するオーバーレイウィンドウ搭載。

- **多言語UI**  
  日本語・英語・中国語・韓国語・ロシア語に対応。

## 💾 インストール方法

1. BOOTHからダウンローダー（`Mio_vrc_download_bundle.zip`）を入手
2. ZIPを展開し、`Mio_vrc_download.exe` を実行
3. インストール先フォルダを選択して「インストール」をクリック
4. 最新版を自動取得してインストール・起動します

> 初回起動時にSenseVoice Smallモデルを自動ダウンロードします（約500MB）。  
> 2回目以降はキャッシュから即時起動します。

---

## 🖥️ 動作環境

- **OS**: Windows 10 / Windows 11（64bit推奨）
- **逆翻訳機能**: Windows WASAPI ループバック対応が必要
- **翻訳機能**: 各翻訳サービスのAPIキーが必要（自己取得）
- **VRChat OSC**: VRChat内のOSC設定を有効にする必要があります

---

## ⚠️ ご注意

- 本ツールは**無償配布**です。転売・有償での再配布は禁止しています。
- 翻訳にかかるAPIの利用料金は利用者ご自身の負担となります。
- 音声データは原則として外部に送信されません（翻訳テキストのみAPIへ送信）。
- VRChat公式ツールではありません。VRChatの利用規約に従ってご使用ください。
